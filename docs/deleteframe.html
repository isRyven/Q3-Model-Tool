Deleting a frame
Deleting frames is, of course, only possible with animated models. An MD3 must have at least 2 frames, before one of them can be removed. If a model contains only 1 frame, it is not possible to remove it.
In this example i've loaded the nicely animated MD3 of the waterpump. The animation has 40 frames (from 0 to 39).
 
 
Stop playback and select the current frame
When deleting a frame, it is important to stop playback, because the current frame will be removed.
Only if You stop playback, You are able to set the current frame with the slider on the 'View'-Tab.
 
 
Removing the frame
Select the choice from the menu:
"Delete current frame..."
 
Confirm the deletion:
 
 
The frame is gone now..
When the selected frame has been deleted, a message on the statusbar will appear.
Also notice that the animation is now shortened by 1 frame, as indicated by the 'End Frame' value right next to the respective slider.
