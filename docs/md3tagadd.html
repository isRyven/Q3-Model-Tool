Adding tags
Tags can be added to an MD3 model. However, the MD3 must not be animated, it only works for non-animated MD3-models.
Tags can be added in two ways: manually add a tag, or add 1 or more tags by importing them from a .tag-file.
To add tags manually, You have to know the exact position of the tag, because You have to type-in this location by hand.
Note: It is not possible to manually add tags with arbitrary axis. Any new tag will be added with a default direction (the standard axis, same axis as in Radiant).
Here i've loaded the ET-truck that has only 2 standard tags, and in this example i will add a new tag on the back of the truck.
 
 
Add tags on the 'Tags'-Tab
Switch to the 'Tags'-Tab, enter a proper name for the tag, and the location of the tag.
Note: Any tag must have a name that has the prefix 'tag_'.
Also note that at this time the model contains 2 tags, as indicated by the tool.
I chose to name my tag: 'tag_myobj'
 
 
Push the button
When You have filled-in the tag-data, press the 'Add'-button right next to the tag-position, to addthe tag to the model.
The tool will now show the new tag-info in the lists. 3 tags now exist in the model.
