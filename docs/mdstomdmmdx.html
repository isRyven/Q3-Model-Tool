Converting .MDS To .MDM/.MDX
To convert a model from MDS-format to an MDM/MDX pair of files, select the proper choice from the menu:
"Load MDS from file..."
 
 
Converting .MDS To .MDM/.MDX
After loading the MDS playermodel, it has already been converted to MDM/MDX.
The statusbar shows a message of the convert.
 
 
Saving the MDM- & MDX-files
Select the choice from the menu to save the file(s): "Save As..."
Note: After loading an MDS, the filtype for saving is automaticly changed to MDM/MDX..
 
 
Saving the MDM
Firstly You are asked to give the new filename of the MDM-file. Supply a name and save the file.
Note: The MDM is the file that contains the model..
 
 
Saving the MDX
Secondly You need to supply the filename of the MDX-file. Enter a name and save the file.
Nore: The MDX is the file that contains the animation of the model..
